To use these environmental encounter tables, first roll a d100 on the frequncy-type table beneath the environment's picture.
Use the result of this roll to determine which of the four frequency-type lists to roll on to find the creature encountered.

The percentages of the frequency-type table are based on "common" = 60%, "uncommon" = 30%, "rare" = 7%, and "very rare" = 3%. Unique creatures are not included when rolling for a random encounter.

These percentages assume each of the frequency types for the environment have the same number of creatures.

Because large differences between the numbers of creatures in each frequency list will skew the actual percentage of an individual creature being encountered,
the actual percentages will likely not match those above.

For an extreme example, if the first three frequency types for an environment have one creature each, but the "very rare" type has 20 creatures, then each of the very rare creatures is going to
be encountered much less often than the intended 3% of the time, and you're going to get bored of seeing that one common and one uncommon creature all of the time.

So, each frequency-type table has been weighted based on the relative number of creatures in each of the environment's frequency-type lists,
with the stipulation that no frequency category will fall below 3%.

When the number of creatures in a frequency-type list cannot be evenly distributed across one of the standerd die sizes,
then the extra slots are distributed across the first creartures in the list, one extra slot each, as needed.

When hovering over a creature's entry, you get a pop-up displaying the creature's size, type, temper, and any tags it has.
Clicking on its link brings you to the creature's page.
