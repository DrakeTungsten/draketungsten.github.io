SKILLS







Skill ranks are bought at level-up. Skill points to spend at level-up are class/level specific, plus INT modifier (but never less than 1)
Purchasing skill ranks:
  • 1 skill point if a class skill
  • 1 skill point if bought w/ INT bonus, regardless if class or non-class skill
  • 2 skill points if bought w/ class level-up skill points and not a class skill
Max rank in any skill = class level + 3
